Using a consistent spacing system makes things look better and work smoother for people using our products. This system allows us to adjust look-and-feels on different devices and lets us change how crowded or spread out the design is, making our products better and easier to use for everyone.

spacing.scale

Scale

Using the 8px base unit as a starting point, our spacing system relies on a specific suite of space values. This scale offers a range from 2px to 80px, each value being a multiple of the base unit. It provides flexibility while ensuring a consistent layout across various screen designs.

spacing.principle

Arrange Items by Similarity

Keep similar items that are related to be grouped together or spaced similarly because it helps users understand how things are connected. When elements have consistent spacing around them, it makes them look similar and helps users see how they're related. For instance, when you have a list or a table, keeping the spacing consistent makes it feel like one complete set of things.

Consistent row spacing signals same-type items

Organise by Closeness

Where things you placed next each other, we think they're connected. Use this idea to show how things are related: put similar stuff closer and less similar stuff further away. For instance, if things are option, keep them close to case they're connected.

Related items are closer; unrelated items have more separation

Establish Structure and Ranking

Hierarchy helps prioritise elements and guides the sequence in which users perceive them. Adjusting the size of elements emphasises larger ones and puts your attention on smaller ones. Likewise, modifying the whitespace around an element can either unify or differentiate elements, affecting their perceived significance.
