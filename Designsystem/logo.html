Specs & Clear Space
Anatomy
1
Mark — Rounded square, accent fill (#FE5104). Contains a white SVG icon at 60% of mark size
2
Wordmark — Product name in 800 weight, stacked above tagline
3
Tagline (optional) — e.g. "Design System", 500 weight, muted colour
4
Clear space — Minimum ½ mark width on all sides. Never crowd the logo
5
Link target — Always wraps an <a href="/"> pointing to the dashboard home

Do / Don't
Do
  • Use the mark on its native accent background
  • Maintain the minimum clear space (½ mark width)
  • Link the logo to the home dashboard
  • Use the on-dark variant on dark/coloured surfaces
  • Use mark-only at small sizes (≤ 28px wordmark becomes illegible)
Don't
  • Change the mark colour or opacity
  • Stretch or distort proportions
  • Place on busy photographic backgrounds
  • Use the light variant on dark surfaces
  • Add drop-shadows or outlines to the mark

function Logo({ size = 'md', surface = 'light', tagline, href = '/', markOnly = false }) {
  const sizeClass = { sm: 'logo-sm', md: 'logo-md', lg: 'logo-lg' }[size];
  const surfaceClass = {
    dark: 'logo-on-dark',
    brand: 'logo-on-brand',
    light: '',
  }[surface];

  const MarkIcon = () => (
    <svg className="logo-mark-inner" viewBox="0 0 20 20">
      <path d="M4 10l4 4 8-8" />
    </svg>
  );

  return (
    <a
      href={href}
      className={['ym-logo', sizeClass, surfaceClass].filter(Boolean).join(' ')}
      aria-label="Yubi Market home"
    >
      <div className="ym-logo-mark" aria-hidden="true">
        <MarkIcon />
      </div>
      {!markOnly && (
        <div className="ym-logo-wordmark">
          <span className="ym-logo-name">Yubi Market</span>
          {tagline && <span className="ym-logo-tagline">{tagline}</span>}
        </div>
      )}
    </a>
  );
}
