# Yubi Market Design System

A comprehensive HTML/CSS design system built for Yubi Market — a B2B financial lending platform. 72 component pages covering everything from foundations to complex interactive components.

## Live Preview

Open `preview.html` in your browser to start browsing.

## What's Included

### Foundations (8 pages)
How to Use, Color System, Typography, Spacing, Grid System, Elevation, Border Radius, Breakpoints

### Components (63 pages)

**Navigation**
Side Navigation, Top Navigation, Breadcrumbs, Tabs, Pagination, Link, Command Menu

**Forms**
Button, Split Button, Input, Textarea, Checkbox, Radio, Choicebox, Toggle, Dropdown, Combobox, Date Picker, Time Picker, Search, Slider, Pin Input, Upload

**Data Display**
Table, List, Avatar, Badge, Tags, Pills, Indicator, Logo, Skeleton, Keyboard Input

**Feedback**
Snackbar, Loader, Progress, Notification, Callout, Stepper, Empty State

**Overlays**
Modal, Drawer, Popover, Tooltip, Bottom Sheet

**Layout**
Accordion, Divider, Filter, FAB, Menu, Icon, Illustration, Grid, Code Block

### Layouts (2 pages)
Side Menu with Navbar, Form Layout

## Tech Stack

- Vanilla HTML + CSS (no build step required)
- CSS custom properties for theming (`data-theme="light|dark"`)
- Each component page is self-contained

## Design Tokens

| Token | Purpose |
|---|---|
| `--bg-panel` | Card / panel background |
| `--bg-inset` | Inset / secondary background |
| `--border` | Border color |
| `--accent` | Primary orange `#FE5104` |
| `--accent-dim` | Accent tint (hover states) |
| `--text-1/2/3` | Primary / secondary / tertiary text |
| `--mono` | Monospace font stack |

## Usage

1. Clone this repository
2. Open any `.html` file directly in a browser — no server required
3. Each component page includes HTML, CSS, and React code examples
4. Download component `.md` files to use as Claude context for code generation
