breakpoints.pattern

Mobile-First Pattern

Always write base styles for the smallest viewport, then layer overrides upward with min-width media queries. This keeps CSS lean and avoids override battles.

/* Correct ✓ — base styles apply to xs, then override up */
.card {
  padding: var(--space-4);
  display: block;
  border-radius: var(--radius-lg);
}

@media (min-width: 768px) {
  .card {
    padding: var(--space-6);
  }
}

@media (min-width: 1024px) {
  .card {
    padding: var(--space-8);
  }
}

/* Wrong ✗ — desktop-first creates needless overrides */
.card {
  padding: var(--space-8);
}
@media (max-width: 767px) {
  .card {
    padding: var(--space-4); /* overriding creates debt */
  }
}
