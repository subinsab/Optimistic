# Empty State — Yubi Market Design System

> **Version:** 1.0.0 | **Category:** Feedback | **Status:** Stable

## Overview

Placeholder content shown when a page, section, or list has no data. Combines an illustration slot, heading, body text, and optional CTA.

## Guidelines

- Write titles as "No [things] found" or "No [things] yet"
- Write descriptions that explain why it's empty and what the user can do
- Provide a CTA only when there's a clear action to resolve the empty state
- Use the compact variant inside cards, drawers, and side panels
