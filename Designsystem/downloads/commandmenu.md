# Command Menu — Yubi Market Design System

> **Version:** 1.0.0 | **Category:** Navigation | **Status:** Stable

## Overview

A searchable command palette overlay. Triggered by ⌘K, it allows quick navigation, action execution, and feature discovery without leaving the current context.

## Structure

- Search bar with icon and Esc hint
- Results grouped by category (Recent, Navigation, Actions)
- Each item: icon + name + optional description + optional keyboard shortcut
- Footer with keyboard navigation hints

## Keyboard behavior

| Key | Action |
|---|---|
| ⌘K / Ctrl+K | Open |
| ↑ / ↓ | Navigate items |
| Enter | Execute selected item |
| Esc | Close |
