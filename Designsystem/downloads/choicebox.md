# Choicebox — Yubi Market Design System

> **Version:** 1.0.0 | **Category:** Forms | **Status:** Stable

## Overview

Card-style selection for mutually exclusive choices (radio behavior) or independent toggles (checkbox behavior). Larger visual footprint than plain radio/checkbox — use for onboarding, settings, and plan selection.

## When to use

- Plan or tier selection (e.g. loan tenures, pricing plans)
- Feature toggles in onboarding flows  
- Any selection where the description adds meaningful context
